# 🎯 Number Guessing Game

A simple interactive number guessing game built using **HTML, CSS, and JavaScript**.

The game generates a random number, and the user tries to guess it within the given attempts. It provides hints based on the user's input and tracks the game status.

---

## ✨ Features

- Random number generation
- User input handling
- Guess validation
- Hint messages:
  - Too high
  - Too low
  - Correct guess
- Attempt tracking
- Restart game functionality
- Interactive user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
